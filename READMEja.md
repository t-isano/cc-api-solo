# 使い方

| API                     | 内容                                            |
| :---------------------- | :---------------------------------------------- |
| GET /characters         | すべてのキャラクター情報を取得する。            |
| GET /characters/:charId | `charId` に紐付いたキャラクター情報を取得する。 |
